Date de mise à jour des données : 2017-09-14

Aperçu des fichiers téléchargables des données normalisées de l'INRP. 

Nom du fichier: NPRI-SubsRele-Normalized-Since1994
Format de fichier : CSV 
Contenu: Le fichier « NPRI-SubsRele-Normalized-Since1994 » contient toutes les quantités de substances rejetées. Pour tout rejet déclaré, les données suivantes sont fournies : année de déclaration (année civile), numéro INRP, nom de l’installation, dénomination sociale de l’entreprise, code du Système de classification des industries de l'Amérique du Nord (SCIAN), province, numéro de registre du Chemical Abstract Service (CAS), nom de la substance, groupe (s’il s’agit d’un rejet dans l’air, l’eau ou au sol ou s’il s’agit de la somme des rejets dans tous les milieux lorsque la quantité déclarée est inférieure à une tonne), catégorie (identifiant le type de rejet, c.-à-d. rejets de cheminée ou ponctuels, émissions fugitives, fuites, autres, etc.), quantité, unités et méthode d’estimation*.

Nom du fichier: NPRI-SubsDisp-Normalized-Since1994
Format de fichier : CSV 
Contenu: Le fichier « NPRI-SubsDisp-Normalized-Since1994 » contient toutes les quantités de substances éliminées sur le site et hors-site, incluant les stériles et résidus miniers. Pour toute élimination déclarée, les données suivantes sont fournies : année de déclaration (année civile), numéro INRP, nom de l’installation, dénomination sociale de l’entreprise, code du Système de classification des industries de l'Amérique du Nord (SCIAN), province, numéro de registre du Chemical Abstract Service (CAS), nom de la substance, groupe (s’il s’agit d’une élimination sur le site ou hors site, de stériles et résidus miniers sur le site ou hors site), catégorie (identifiant la méthode d’élimination, c.-à-d. enfouissement, injection souterraine, etc.), quantité, unités et méthode d’estimation*.

Nom du fichier: NPRI-SubsDisp-Tran-Recy-Normalized-Since1994
Format de fichier : CSV 
Contenu: Le fichier « NPRI-SubsDisp-Tran-Recy-Normalized-Since1994 » contient toutes les quantités de substances transférées aux fins de recyclage ou de traitement hors site antérieur à l’élimination finale.  Pour tout transfert déclaré, les données suivantes sont fournies : année de déclaration (année civile), numéro INRP, nom de l’installation, dénomination sociale de l’entreprise, code du Système de classification des industries de l'Amérique du Nord (SCIAN), province, numéro de registre du Chemical Abstract Service (CAS), nom de la substance, groupe (s’il s’agit d’un transfert aux fins de recyclage ou pour traitement précédant l’élimination finale), catégorie (identifiant la méthode de traitement ou de récupération, c.-à-d. traitement chimique, traitement biologique, récupération de solvants, etc.), quantité, unités et méthode d’estimation*.

Nom du fichier: NPRI-Substance-Comments-Normalized-Since1994
Format de fichier : CSV
Contenu: Le fichier « NPRI-Substance-Comments-Normalized-Since1994 » contient tous les commentaires fournis par le déclarant à propos des substances indiquées dans le rapport. Pour tout commentaire, les données suivantes sont fournies : année de déclaration (année civile), numéro INRP, nom de l’installation, dénomination sociale de l’entreprise, code du Système de classification des industries de l'Amérique du Nord (SCIAN), province, numéro de registre du Chemical Abstract Service (CAS), nom de la substance, code pour le type de commentaire (nom raccourci du type de commentaire), type de commentaire (par exemple, raison de l’ajout de cette substance par l’installation, commentaires sur les éliminations, etc.) et commentaires.

Nom du fichier: NPRI-Facility-Geo-Locations-Since1994
Format de fichier : CSV / XLSX / JSON
Contenu: Le fichier « NPRI-Facility-Geo-Locations-Since1994 » contient toute information reliée à l’identification de l’emplacement de toutes les installations ayant déclaré à l’INRP. Pour toute installation ayant déclaré à l’INRP depuis la mise en vigueur du programme, les données suivantes sont fournies : année du plus récent rapport déposé (année civile), numéro INRP, raison sociale et nom de l’installation, adresse, ville, province, code postal, description de l’arpentage, description topographique nationale, latitude, longitude, datum, secteur industriel clé, etc. D’autres données géographiques sont fournies dans ce fichier.
 

*Méthode d'estimation: 
M1 - surveillance en continu des émissions (SCE)
M2 - contrôle prédictif des émissions (CPE)
M3 - analyse à la source
C - Bilan massique
E1 - facteurs d'émissions propres à l'installation
E2 - facteurs d'émissions publiés
O - Évaluations techniques
NI - Aucune information disponible
NA - Sans objet

Overview of the contents of the downloadable NPRI Bulk Data files. 
	
File Name: NPRI-SubsRele-Normalized-Since1994
File Format: CSV 
Contents: NPRI-SubsRele-Normalized-Since1994 contains all substance release quantities. For each release, the following data is provided: Reporting Period (calendar year), NPRI ID, Facility Name, Company Name, North American Industry Classification System (NAICS) code, Province Name, Chemicals Abstract Service (CAS) Registry Number, Substance Name, Group (identifies if release was to air, land, water, or reported as 'sum of all media' when the release amount is less than 1 tonne), Category (identifies the type of release, i.e. stack/point, fugitive, leaks, other, etc.), Quantity, Units, and Estimation Method*. 

File Name: NPRI-SubsDisp-Normalized-Since1994
File Format: CSV 
Contents: The NPRI-SubsDisp-Normalized-Since1994 file contains all on-site and off-site disposal quantities, including Tailings and Waste Rock (TWR). For each disposal, the following data is provided: Reporting Period (calendar year), NPRI ID, Facility Name, Company Name, North American Industry Classification System (NAICS) code, Province Name, Chemicals Abstract Service (CAS) Registry Number, Substance Name, Group (identifies if disposal was on-site, on-site TWR, off-site, or off-site TWR), Category (identifies the method of disposal, i.e. landfill, underground injection, etc.), Quantity, Units, and Estimation Method*. 

File Name: NPRI-SubsDisp-Tran-Recy-Normalized-Since1994
File Format: CSV
Contents: The NPRI-SubsDisp-Tran-Recy-Normalized-Since1994 file contains all quantities transferred for recycling or treatment prior to disposal. For each transfer, the following data is provided: Reporting Period (calendar year), NPRI ID, Facility Name, Company Name, North American Industry Classification System (NAICS) code, Province Name, Chemicals Abstract Service (CAS) Registry Number, Substance Name, Group (off-site recycling or off-site treatment), category (identifies the method of treatment or recovery, i.e. chemical treatment, biological treatment, solvent recovery, etc.), Quantity, Units, and Estimation Method*. 

File Name: NPRI-Substance-Comments-Normalized-Since1994
File Format: CSV 
Contents: The NPRI-Substance-Comments-Normalized-Since1994 file contains all the reporter-provided comments about substances included in their report. For each comment, the following data is provided: Reporting Period (calendar year), NPRI ID, Facility Name, Company Name, North American Industry Classification System (NAICS) code, Province Name, Chemicals Abstract Service (CAS) Registry Number, Substance Name, Comment Type Code (Short form of Comment Type Name), Comment Type Name (ex. Reason Why Facility Is Adding This Substance, Disposal Comments, etc.), and Comment.

File Name: NPRI-Facility-Geo-Locations-Since1994
File Format: CSV / XLSX / JSON
Contents: The NPRI-Facility-Geo-Locations-Since1994 file contains complete location information for all facilities that have reported data to the NPRI. For each NPRI reporting facility since the inception of the program, the following data is provided: year of last filed report, NPRI ID, company and facility name, address, city, province, postal code, physical land survey description, physical national topographical description, latitude, longitude, datum, key industrial sector, etc. Other geographical data is provided in this file.

* Estimation method codes: 
M1 - Continuous Emission Monitoring
M2- Predictive Emission Monitoring
M3 - Source Testing
C - Mass Balance
E1 - Site Specific Emission Factors
E2 - Published Emission Factors
O - Engineering Estimates
NI - No Information Available
NA - Not Applicable
